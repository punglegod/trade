# TradeReplayAssistant v0.1.0

发布日期：2026-04-16  
发布类型：MVP 基线版本

## 本版目标

- 交付可用 MVP：`记录 + 统计 + 周报`
- 完成 Liquid Glass + 药丸风格视觉体系首版落地
- 建立可复用的 GitHub Actions iOS CI 流程

## 核心能力

1. 交易记录
- 新增/编辑/删除交易
- 按标的与策略标签筛选
- 本地持久化与远端同步（LWW 冲突策略）

2. 统计看板
- 支持 `近7天 / 近30天 / 年内` 时间窗
- 指标包含：总笔数、胜率、总盈亏、平均盈亏、Profit Factor、最大回撤

3. 周报生成
- 自动生成：本周做得好、待改进、下周行动项
- 无交易数据时返回默认引导内容

4. 视觉系统
- iOS 26+：原生 Glass 效果与玻璃按钮样式
- iOS 17-25：材质降级保持一致布局与交互结构
- 胶囊组件统一应用于筛选、标签、CTA、分段切换

## 工程与 CI

- GitHub Actions：自动 `build + test`（iOS Simulator）
- 修复了测试环境下的主线程隔离与链接配置问题
- 测试运行时使用内存存储，降低 CoreData 路径探测噪声
- 固定 CI simulator 架构为 `arm64`，减少 destination 匹配警告

## 已验证

- 单元测试通过：
  - `ReportServiceTests`
  - `TradeRepositoryTests`
- 回归覆盖主路径：
  - `新增交易 -> 统计更新 -> 周报生成 -> 删除后统计回落`

## 已知限制

- 仍需在真实设备或本地 Xcode 中完成 UI 体验走查（滚动性能、视觉层级、交互动效）
- 图标、隐私清单、App Store 元数据仍需补齐后再走外部测试分发
